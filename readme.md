> Caputre mobile tweet


## Install

```
$ npm install capture-mobile-tweet
```


## Usage

```js
const fs = require('fs');
const cap = require('capture-mobile-tweet');

const tweet = 'https://mobile.twitter.com/akameco/status/904080083802923008'

cap(tweet).then(img => {
  fs.writeFileSync('mobile-tweet.png', img)
})
```
